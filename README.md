# Conference

Веб-приложение для управления конференциями, спикерами и профилем пользователя. Проект собран на Vue 3, Vite и Pinia.

## Возможности

- список конференций с фильтрацией и просмотром деталей
- таймер до начала конференции
- бронирование и напоминания
- страницы регистрации, входа, профиля и расписания
- локальное хранение данных в браузере

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Примечание

Проект работает без отдельного backend-сервера. Данные сохраняются локально в браузере.

## Подключение к backend

Если backend уже готов, добавьте файл `.env` в корень проекта и укажите:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

После этого фронтенд начнет отправлять запросы в backend, а localStorage останется запасным режимом.
